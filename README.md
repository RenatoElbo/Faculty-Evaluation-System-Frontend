# Intelligent Faculty Evaluation System — Frontend

 A smart faculty evaluation platform that collects student evaluations and uses **AI (Machine Learning + NLP)** to detect inconsistencies between ratings and written feedback/comments.

 ⚠️ **Note:** Questionnaire Page and Generate Results feature are **not yet fully implemented** (~90% done) as this is currently being restructured.

## Tech Stack
- **React + TypeScript** — UI & type safety
- **Vite** — Build tool
- **Tailwind CSS** — Styling

## Features
- 📝 Student evaluation submission
- 🤖 AI-powered analysis — detects misalignment between ratings and comments using ML & NLP
- 📊 Admin dashboard for monitoring evaluation results
- 👨‍🏫 Faculty information management
- 🔐 Admin account system

## Getting Started
```bash
# Install dependencies
npm install

# Run development server
npm run dev
```
